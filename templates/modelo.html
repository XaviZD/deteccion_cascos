Ultralytics YOLOv8

El modelo por el cual funciona nuestro sistema de detección es YOLOv8, la última versión del famoso sistema de detección de objetos, fue entrenado teniendo como base dos set de datos compuestos de personas en ambientes laborales usando cascos de seguridad y otros casos de personas sin dicho implemento de seguridad, para darle una mayor confiabilidad al sistema, el segundo dataset sirve para confirmar los valores.
